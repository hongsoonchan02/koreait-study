1) 텍스트 요소

텍스트를 표현하거나 의미를 부여하는 태그 종류들

<h1> ~ <h6> 제목

설명 : 제목을 나타내는 태그이며 숫자가 커질수록 중요도가 낮음

h6 제목 예시

h2 제목 예시



<strong> <b> 문자 강조

strong 설명 : 중요하거나 강조되어야 하는 내용을 나타냄

b 설명 ㅣ 중요하거나 강조되어야 하는 내용을 시각적으로만 나타냄

차이점 : strong은 스크린 리더 등의 사용자에게 중요한 내용임을 알려줌

strong 입니다. b 입니다.

<small> 작은 문자

설명 : 기본 문자보다 작게 표시하는 태그

small 입니다.

<u> <s> 밑줄과 취소선

u 설명 : 글자에 밑줄을 표시하는 태그

s 설명 : 글자에 취소선을 표시하는 태그

u 입니다. s 입니다.

<p> 단락

설명 : 글의 단락(문단)을 나타내는 태그

p 입니다.

<sup>, <sub> 첨자

sub 설명 : 위 첨자로 나타내는 태그

sub 설명 : 아래 첨자로 나타내는 태그

sup 입니다. sub 입니다.

2) 이미지와 하이퍼링크 요소

이미지와 하이퍼링크를 사용하는 태그 종류

<img> 이미지

설명 : 이미지를 보여주는 태그

<a> 하이퍼링크 태그

설명 : 다른 웹페이지, 파일. 이메일 주소 등으로 연결하는 링크를 생성하는 태그

네이버로 이동하기

이미지와 하이퍼링크 같이 사용하기

설명 : a 요소 안에 img 요소를 넣어서 이미지를 클릭하면 이동되도록 사용 가능합니다.

3) 목록 요소

목록들을 나타내는 태그 종류

<ul>, <ol>, <li>

ul 설명 : 순서 없는 목록

ol 설명 : 순서 있는 목록

li 설명 : 항목

순서 없는 목록
  • HTML
  • CSS
  • JavaScript
순서 있는 목록
  1. HTML
  2. CSS
  3. JavaScript

<dl>, <dt>, <dd> 정의 목록

dl 설명 : 정의 목록 태그

dt 설명 : 용어를 나타내는 태그

dd 설명 : 설명을 나타내는 태그

테이블 요소

엑셀처럼 테이블을 표현할 수 있는 태그

<table>, <tr>, <th>, <td>

table 설명 : 웹 페이지에서 테이블을 만들 때 사용

tr 설명 : 한 개의 행을 만드는 태그

th 설명 : 행에 데이터를 넣을 때 사용되는 태그 (제목 용도)

td 설명 : 행에 데이터를 넣을 때 사용되는 태그
